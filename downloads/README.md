# Template Zip Files

This directory should contain zip files for each template. The zip files should be named according to the following pattern:

- For Portfolio templates: `Portfolio_1.zip`, `Portfolio_2.zip`, etc.
- For Restaurant templates: `Resturant_template_1.zip`, `Resturant_template_2.zip`, etc.
- For Textile templates: `Textile_template_1.zip`, `Textile_template_2.zip`, etc.

## How to create zip files:

1. Navigate to each template directory (e.g., `Portfolio_1`)
2. Select all files in the directory
3. Right-click and choose "Send to" > "Compressed (zipped) folder"
4. Rename the zip file to match the directory name
5. Move the zip file to this `downloads` directory

## Example:

For `Portfolio_1`:
1. Go to `Portfolio_1` directory
2. Select all files
3. Right-click > Send to > Compressed (zipped) folder
4. Rename the zip file to `Portfolio_1.zip`
5. Move `Portfolio_1.zip` to this `downloads` directory
